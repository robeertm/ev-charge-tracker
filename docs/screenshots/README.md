# Screenshots

Drop your screenshots into this folder with these filenames so they appear in the main README:

- `dashboard.png` — main dashboard with KPI cards, charts, and live vehicle widget
- `dashboard-dark.png` — dashboard in dark mode
- `input.png` — new charge form with Start/Stop buttons
- `history.png` — charge history with filters
- `settings.png` — settings page (or just the Vehicle API section)
- `report.png` — first page of the generated PDF report

Recommended size: 1200-1600 px wide. PNG or JPG.
